Description détaillée

Classe pour la fenêtre principale de l'application.

Cette classe est la classe exécutant la fenêtre principale de l'application, elle permet d'avoir accès a la liste de contacts. Elle permet également d'ouvrir les différentes fenêtres et de montrer les logs de l'application.
